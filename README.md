** 注意: 开发中, 请勿用于生产环境. **

## 使用方法

- 克隆代码

```
$ git clone https://github.com/laispace/pager.git
```

- 安装依赖

```
$ cd pager
$ npm install
```

- 开启 mongodb

```
$ sudo mongod
```

需要提前创建好名为 pager 的数据库, 并添加帐号为 pager 密码为 pass4pager 的用户.

- 开发调试

```
$ npm run start
```

- 构建

```
$ npm run build
```
